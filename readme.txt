Proyecto VivaQuiva:

- VivaQuiva es una pyme cordobesa, la cual tiene como objetivo brindar servicios de diseño industrial y de indumentaria en varios rubros.
- A lo largo del desarrollo del proyecto realice muchas modificaciones, tuve que rehacer la página varias veces para poder aplicar las nuevas herramientas, lo que fue un desafío para mí.
- Investigué para poder implementar otras utilidades tales como JS.
- Las medias querys están pensadas para Mobile, Tablet y Monitores.
- Otro desafío muy grande fue que las imágenes que me brindó mi "cliente" están en muy mala calidad, son fotos viejas, las cuales fueron muy difíciles de trabajar.
